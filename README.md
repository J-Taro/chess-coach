# Chess Coach ♟️

> 一个面向入门新手的国际象棋 AI 教练，帮助你理解每一步棋的好坏和背后的原因。

如果你已经知道每个棋子怎么走的基本规则，但不明白"为什么不能走这里"、"这步棋哪里出了问题"、"对手在威胁什么"——Chess Coach 正是为你设计的。

AI 教练不只告诉你最佳走法，而是**用新手能听懂的语言解释原因**，帮你建立真正的棋感。

## 功能

- **实时双人对战**：创建房间、输入房间码邀请朋友，或快速匹配对手
- **选择执棋颜色**：创建房间时可选白方、黑方或随机分配
- **单人练习模式**：自己操控双方，随时练习走法
- **FEN 局面分析**：输入任意 FEN 字符串，直接获取 AI 局面解读
- **AI 教练分析**：一键获取当前局面分析，站在你的角度给出建议
- **经典策略识别**：AI 自动识别开局、战术模式并标注策略名称
- **完整走法历史**：AI 分析时参考整局走法，给出更准确的建议
- **断线重连**：对手断线后房间保留，重新输入房间码即可继续
- **合法走法高亮**：拖动棋子时显示可落子位置

## 技术栈

- 后端：Python + Flask + Flask-SocketIO
- 前端：chessboard.js + chess.js + Socket.io
- AI：豆包大模型（字节跳动火山引擎）

## 快速开始

**1. 安装依赖**

```bash
pip install flask flask-socketio openai python-chess
```

**2. 配置 API Key**

新建 `config.py`，填入你的火山引擎 API Key：

```python
VOLC_API_KEY = "你的真实key"
```

**3. 启动**

```bash
python3 app.py
```

打开浏览器访问 http://127.0.0.1:5000

## 使用方法

**双人对战**
1. 一方创建房间，选择执棋颜色
2. 另一方输入房间码加入，或使用快速匹配
3. 拖动棋子走棋，拖动时显示合法落点
4. 任意时刻点击「🎓 教练分析」获取 AI 建议

**单人练习**
1. 点击「单人模式」直接开局
2. 自己操控白黑双方，自由练习

**FEN 分析**
1. 从 Lichess 等平台复制 FEN 字符串
2. 粘贴到首页 FEN 输入框，点击分析
3. 获取当前局面的形势评估和建议
